mac下安装erlang

发表于 2016-08-19   |   分类于 技术

  • 下载erlang源码

    地址http://www.erlang.org或者https://github.com/erlang/otp

  • 将源码进行解压缩

    tar -zxf otp_src_17.4.tar.gz

  • 进入到解压缩的文件夹

    cd otp_src_17.4

  • 配置环境变量

    vi ~/.bash.profile

    export ERL_TOP=pwd

    （pwd是获取当前的绝对路径，注意一定要是erlang源码的地址）

    export LANG=C

  • 编译与安装

    make

    sudo make install

  • 安装目录是：/usr/local

#erlang
